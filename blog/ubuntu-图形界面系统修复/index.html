ubuntu错误修改系统配置文件

  • 问题描述：

    各种无法开机，无法进入图形界面，或者各种修复方法失效，这是一个不错的尝试。

  • 命令描述
$ sudo rm /var/lib/apt/lists/lock
$ sudo rm /var/lib/dpkg/lock
$ sudo rm /var/lib/dpkg/lock-frontend
$ sudo dpkg --configure -a
$ sudo apt clean
$ sudo apt update --fix-missing
$ sudo apt install -f
$ sudo dpkg --configure -a
$ sudo apt upgrade
$ sudo apt dist-upgrade

最后重启系统：

$ sudo reboot
