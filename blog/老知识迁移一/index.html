清理~！！！！
1、把安装包里的data文件夹删除

第一步：下载mysql5.7的压缩文件，解压到你想要的地址
如：D:\mysql57

第二步：在与bin同目录下新建my.ini(新建记事本更改后缀即可)
配置你的mysql
如：[mysql]

设置mysql客户端默认字符集 
 default-character-set=utf8  
 [mysqld] 
 
 #设置3307端口 
port=3307  
    
设置mysql的安装目录 
basedir="D:\mysql57" 
    
设置mysql数据库的数据的存放目录 
datadir="D:\mysql57\data"
    
服务端使用的字符集默认为8比特编码的latin1字符集  
character-set-server = utf8

创建新表时将使用的默认存储引擎
default-storage-engine=INNODB

允许最大连接数
max_connections=200
注：data文件夹无需创建

第三步：管理员身份进入cmd 并进入D:\mysql57\bin 安装mysql
输入 mysqld -install mysqlXXX
注：因为是电脑上的第二个MySQL  所以用“XXX”（内容自定）区别与之前服务 

第四步：mysqld --initialize 初始化  这是data文件会自动创建

第五步：最重要的一步注册列表，在如下地址中找到mysqlXXX
计算机\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\mysql3307
编辑imagepath 将地址改为你安装的新的MySQL的地址。。。。。

3.3redirct和reverse的使用

# user/logout
class LogoutView(View):   
    """退出登录"""    
    def get(self, request):       
        """退出登录"""        
        # 清除用户的session信息        
        logout(request)        
        # 跳转到首页        
        return redirect(reverse('goods:index'))

3.4正则路由参数的传递和接收

  • re_path((\d+),views.re_handler)

    正则表达式匹配原则

    表达式 说明
    . 除了\n和\r的所有字符
    \d 数字
    \D 非数字
    \w 数字字母和下划线
    \W 非。。
    \s 空格包括制表符，换页符等*
    * 出现次数大于等于0
    + 出现次数大于等于1
    {n} 出现n次
    {m，n} n>=出现次数>=m
    ^ 以开头
    $ 以结尾

4.2创建模型

#对象类描述
class Address(BaseModel):
    '''地址模型类'''
    user = models.ForeignKey('User',verbose_name='所属账户',on_delete=models.CASCADE)
    receiver = models.CharField(max_length=20,verbose_name='收件人')
    addr = models.CharField(max_length=256,verbose_name='收件地址')
    zip_code = models.CharField(max_length=6,null=True,verbose_name='邮政编码')
    phone = models.CharField(max_length=11,verbose_name='联系电话')
    is_default = models.BooleanField(default=False,verbose_name='是否默认')

    objects = AddressManager()

    class Meta:
        db_table = 'df_address'
        verbose_name = '地址'
        verbose_name_plural = verbose_name
