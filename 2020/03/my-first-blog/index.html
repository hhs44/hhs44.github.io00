My First Blog

Mar 9, 2020 by Huang ao
标签: demo hugo theme
分类: Development golang

对主题的简单了解与处理

此处使用的主题为hugo-theme-bootstrap4-blog

简单初始化后的见面如下，其中home，about，contact的页面内容直接使用 hugo new name.md 即可进行内容的填充。posts则为自己自定义的文章分类。

用法

将存储库克隆到您站点的themes目录。

然后修改 config.toml 或 config.yaml 增加一行

theme = "主题文件夹的名称"

内容的建议

  • 将博客文章保留在content/posts目录中，例如：content/posts/my-first-post.md

  • 将静态页面保留在content目录中，例如：content/about.md

  • 将类似图像的媒体保留在static目录中，例如：static/2016/10/screenshot.png

  • 如果您希望在社交媒体上分享帖子时显示图片，请在帖子的首页中至少指定一张图片，例如： images: ["/2016/10/screenshot.png"]

  • 使用<!--more-->帖子中的标签控制摘要页面上显示的帖子数量

  • 通过设置帖子的comments = false前沿禁用评论

  • 通过设置禁用整个站点（或单个页面/帖子）上的社交共享图标 sharingicons = false

  • 如果您的内容存储在git中，请添加enableGitInfo = true到您的网站配置中，Hugo将使用git历史记录在页面元数据中设置更准确的修改日期
